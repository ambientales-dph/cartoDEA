# **App Name**: CartoDEA v4.1

## Core Features:

- Base Map Display: Display the base map using OpenLayers, integrating data from Google Earth Engine.
- Geospatial Analysis: Perform geospatial analysis using Turf.js for operations like buffering, intersection, etc.
- Data Overlay: Overlay custom data layers on the map.
- AI-Powered Feature Suggestions: Suggest relevant geospatial features to display based on user context and preferences, leveraging Genkit as a tool to assist in deciding what to display.
- User Interaction: Enable map manipulation (zoom, pan) and feature selection.

## Style Guidelines:

- Primary color: Deep sky blue (#4169E1), reminiscent of classic maps and charts.
- Background color: Light gray (#F0F0F0), providing a neutral backdrop for the map data.
- Accent color: Forest green (#228B22), emphasizing data features, analysis results and actionable elements.
- Font: 'Inter', a sans-serif font for clear and modern data display. Use for both headline and body text.
- Use clean and recognizable icons from Lucide React for map controls and feature representations.
- A clean and intuitive layout using ShadCN UI components, optimized for map-centric interaction.
- Subtle animations for data loading and map transitions.
